# krach-generator

## Requirements
* apt-get install python3 python-venv

For Raspberry pi support:
* apt-get install install python-rpi.gpio python3-rpi.gpio

## Installation
* make venv

## Run
* source .venv/bin/activate
* cd krach-generator/lib/
* ./krgen.py -s ../../sounds/

## Exit
* deactivate

## Systemd integration
* see the systemd folder
